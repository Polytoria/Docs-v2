# :polytoria-UILabel: UILabel

{{ inherits("UIView") }}

UILabel is a class that can be used to display text.


## Properties

### Text:string { property }

The text of the label.

---
### TextColor:Color { property }

The color of the text.

---
### OutlineWidth:number { property }

The width of the text outline.

---
### OutlineColor:Color { property }

The color of the text outline.

---
### HorizontalAlignment:TextHorizontalAlignmentEnum { property }

Determines the horizontal alignment of the text.

---
### VerticalAlignment:TextVerticalAlignmentEnum { property }

Determines the vertical alignment of the text.

---
### FontSize:number { property }

The font size of the text.

---
### UseRichText:boolean { property }

Determines whether the text uses rich text formatting.

---
### FontAsset:FontAsset { property }

The font asset used for the text.
